Professional Projects

Game development: A Little Lost (2015-2016)

Free-to-play match-3 puzzle game released by Goodgame Studios in August 2016 for Facebook, iOS and Android. I worked on the project as a client developer.

The web version of the game is on Flash, using libraries such as Starling and Robotlegs. The mobile version is built on Unity. Both versions use Spine for animations. I worked on the project as a client developer from its early stages all the way to the global release.

University Projects

Information visualization: Master's Thesis (2013)

Master's thesis examining the theoretical and practical process of exploratory information visualization, and the development of two special-purpose visualizations.

The thesis is titled "Exploratory Visualization of Inter-Organizational Networks." The project included research into the process of exploratory visualization as well as the iterative development of two highly specialized visualizations for Aalto University's Institutional Relationships unit. The visualizations were built with Processing. User interviews were central to the development of the visualizations to ensure they are genuinely relevant to end users.

  • thesis (pdf)

Interaction design: Block game (2012)

Multitouch / gestural interaction project on the course Experimental User Interfaces with Kenta Kusumoto and Antti Seppä.

The result is a cooperative team game of creating simple shapes and "throwing" them to the opposing team's side. For most shapes, creation requires more than two hands. The game is built with Processing and the TUIO framework for multitouch devices.

  • video

Interaction design: 3D Pong (2012)

Computer vision project on the course Experimental User Interfaces with Pyry Kröger and Ville Hakola.

The squash-like game uses Kinect to recognize players without any additional equipment, so it can played using barehands to control the "rackets". The game is built with Processing and a few frameworks (Simple OpenNI) for recognizing the players.

  • video
  • source @Github

Game design: Murderbook (2010)

Murderbook was a detective-story inspired attempt by myself, Mikael Matveinen, Rauno Järvinen, Marko Tyrväinen and Anton Zvonkov to create a genuinely social game utilising a social media platform.
